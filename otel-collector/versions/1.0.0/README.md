# OpenTelemtry Collector
The OpenTelemetry Collector is a tool for receiving, processing, and exporting telemetry data such as traces and metrics using the OpenTelemetry Protocol (OTLP).

### Warning

- This template does not create a GVC, and you will need to turn on tracing on the GVC level after installing, specifying the target workload and port. This will trigger a restart of all workloads in the GVC.
